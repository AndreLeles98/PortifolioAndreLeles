Key Contributions

  • Developed a real-time level editor with seamless transition between editing and gameplay modes for rapid iteration and playtesting
  • Implemented core editor systems, including grid-based placement aligned to gameplay rhythm and dynamic content insertion

  • Built selection tools supporting multi-selection, movement, and duplication workflows

  • Implemented multi-input support (keyboard, mouse, and gamepad) for full editor interaction
  • Added real-time visualization modes (silhouette, in-game view, world style variations)
  • Contributed to save system integration for editor data and workflows

Key Features

  • Implemented full transformation pipeline (model, view, projection matrices)
  • Developed custom OBJ loader and binary exporter for optimized data access
  • Built triangle rasterization pipeline with pixel-level control
  • Implemented Z-buffer for depth testing
  • Implemented back-face culling and frustum culling
  • Implemented perspective-correct interpolation

  • Handled basic clipping and screen-space transformations
